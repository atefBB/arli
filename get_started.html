قبل كل شيء...

لتعلم كبداية بأنك تتعامل مع لغة البرمجة JavaScript، مما يتيح لك إستعمال آرلي في عدةمجالات:

المتصفح

أولا يجب أن تحمل الملف الخاص بالإنتاج أو التطوير، ملف الإنتاج سريع، أما ملف التطوير ثقيل لكنه يحتوي على النعليقات التي تساعدك في التطوير.
تحميل ملف الإنتاج
تحميل ملف التطوير
ثم يمكننا إدراج الملف مع صفحة HTML: 


<script src="arli.js"></script>
<!-- أو -->
<script src="arli.min.js"></script>
<!-- أو من الموقع بدون تحميل الملف-->
<script src="https://raw.githubusercontent.com/elkebirmed/arli/master/dist/arli.min.js"></script>

Bower

يمكنك تنصيب آرلي من خلال Bower: 


$ bower install arli

ثم تستعمله في صفحاتك: 


<script src="bower_components/arli/dist/arli.min.js"></script>

npm

يمكنك تنصيب آرلي من خلال npm على منصة Node.js أو io.js من خلال إحدى الطرق التالية: 


$ npm install arli
$ npm install arli --save
$ npm install arli --save-dev
$ npm install arli --global

ثم تستعمله في برنامجك: 


var arli = require('arli');
// أو باستعمال طريقة جافاسكريبت 6
import arli from 'arli';

أنت جاهز الآن

بعد أن نصبت آرلي بطريقة أو بأخرى يمكنك إستعمال البرنامج بكل بساطة: 


arli.transform('مرحبا بالعـــــــــــالم');
// -> مرحبا بالعـالم

arli.transform('0123456789');
// -> '٠١٢٣٤٥٦٧٨٩'

arli.transform('0123456789', {digit: false});
// -> '0123456789'
